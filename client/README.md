# poke-game-front-ts

## Setup

```bash
yarn
yarn serve
```

## 环境变量

```env
VUE_APP_ENABLE_MULTIPLE_SOCKET
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

## Project structure
```

    ├─components

    ├─service     // api

    ├─utils
    └─views
```

# poke-game-front-ts

## Setup

```bash
yarn
yarn serve
```

## 环境变量

```bash
VUE_APP_API_IP
VUE_APP_API_PORT
```

### 编译

```bash
VUE_APP_API_IP=1.2.3.4 VUE_APP_API_PORT=1111 NODE_ENV=production yarn build
```

### Run your tests

```bash
yarn run test
```
